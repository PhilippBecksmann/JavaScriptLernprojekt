# 📚 JavaScript Lernprojekt

> Strukturiertes Lernprojekt begleitend zu einem Udemy-Kurs –
> mit Lernmaterialien, Karteikarten und praktischen Übungen.

## 📖 Bisher gelernte Themen
- Strict Mode
- Variablen: var, let, const
- Datentypen & Operatoren
- Kontrollstrukturen (if/else)
- Funktionen
- Objekte mit Methoden & this
- Template Literals
- Arrays
- Schleifen (for, while u.a.)
